Их нужно выполнить, чтобы получить зачёт и диплом о профессиональной переподготовке.Задачи со звёздочкой (*) — это дополнительные задачи и/или задачи повышенной сложности. Их выполнять не обязательно, но они помогут вам глубже понять тему.Домашнее задание выполните в файле readme.md в GitHub-репозитории.
